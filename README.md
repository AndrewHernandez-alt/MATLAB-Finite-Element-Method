# MATLAB-Finite-Element-Method
MATLAB Finite Element Method (for static structures with input frequencies)
Developed from scratch a MATLAB program for structural
vibration analysis using FFT techniques to identify dominant frequencies in steel structures. Implemented global stiffness
matrix assembly, modal analysis, and node-level stress estimation for structural evaluation.

NOTE: You need to have the excel file in the same folder. Also be sure to download the findpeak toolbox.
